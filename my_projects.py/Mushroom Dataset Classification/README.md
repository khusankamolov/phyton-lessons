# 🍄 Mushroom Classification using Machine Learning

## 📌 Overview

This project presents an end-to-end machine learning pipeline for classifying mushrooms as **Edible** or **Poisonous** using the Mushroom Classification dataset.

The workflow covers every major step of a classification project, including data preprocessing, exploratory data analysis (EDA), model training, evaluation, and performance comparison across multiple machine learning algorithms.

---

## 🎯 Objectives

* Perform data preprocessing and cleaning
* Explore the dataset through visualizations (EDA)
* Train multiple classification models
* Compare model performance using evaluation metrics
* Identify the best-performing algorithm for mushroom classification

---

## 🤖 Machine Learning Models

The following classification algorithms were implemented and evaluated:

* Decision Tree
* Random Forest
* K-Nearest Neighbors (KNN)
* Logistic Regression
* Support Vector Machine (SVM)
* XGBoost

## 📊 Model Evaluation

The models were evaluated using:

* Precision
* Recall
* F1-Score
* Confusion Matrix

# 🏦 Loan Approval Prediction using SVM

This project predicts loan approval status using Machine Learning (SVM classifier).

## 📊 Dataset
Columns:
- Loan_ID
- Gender
- Married
- Dependents
- Education
- Self_Employed
- ApplicantIncome
- CoapplicantIncome
- LoanAmount
- Loan_Amount_Term
- Credit_History
- Property_Area
- Loan_Status (Target)

## ⚙️ Steps
1. Data preprocessing
2. Handling missing values
3. Encoding categorical features
4. Feature scaling (StandardScaler)
5. Train-test split
6. Model training (SVM)
7. Evaluation

## 🤖 Model
- Algorithm: Support Vector Machine (SVM)
- Kernel: Linear
- Accuracy: ~83%

## 📈 Result
The model predicts loan approval based on applicant profile with good performance.

- Pandas

- Matplotlib

## 🎯 Objective
To build a machine learning model that helps automate loan approval decisions.
